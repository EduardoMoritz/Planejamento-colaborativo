## Criação das tarefas 

### *Ordem de prioridade*

x1. Criação "Tela inicial"

x2. Função "Tela inicial" para que o usuário tenha acesso as diversas funções do aplicativo

x3. Design da tela inicial

x4. Criação aba "Configurações"

x5. Função "Configurações" para que o usuário tenha acesso as configurações do aplicativo

x6. Criação aba "Minha conta"

x7. Função "Minha conta" para o usuário entrar com sua conta google ou etc.

x8. Criação aba "Tarefas"

x9. Função "Tarefas" para o usuário ver seus afazeres

x10. Função "Editar tarefas" para que o usuário possa alterar informações de suas tarefas

x11. Design do interior do aplicativo

12. Criação aba "Reuniões"

13. Função "Reuniões" para que o usuário possa marcar reuniões importantes em seu calendário

14. Função "Notificações" para que o usuário seja informado de suas tarefas pendentes ou reuniões

15. Criação aba "Agenda"

16. Função "Agenda" para que o usuário possa realizar anotações

17. Função "Compartilhar" para que o usuário possa compartilhar suas informações do aplicativo

18. Criação "Banco de Dados"

19. Design do ícone do aplicativo

20. Criação aba "Atividades em conjunto"

21. Função "Atividades em conjunto" para que o usuário possa marcar tarefas, reuniões e anotar coisas com outras pessoas

22. Criação aba "Central de ajuda"

23. Função "Central de ajuda" para que o usuário possa entrar em contato em caso de dúvidas sobre o uso do aplicativo

24. Criação aba "Conta premium"

25. Função "Conta premium" para que o usuário tenha acesso a mais ferramentas dentro do aplicativo e não receba anúncios 

26. Criação aba "Pagamentos"

27. Função "Pagamentos" para que o usuário possa conferir dados de pagamento, quando ira vencer etc

28. Função "Anúncios" para que os usuários recebam anúncios e a plataforma consiga lucrar em cima

29. Função "Bloquear AntiAD" para que o usuário não utilize ferramentas que bloqueiam anúncios

30. Criação aba "Criar modelo"

31. Função "Criar modelo" para que o usuário tenha acesso a diversos modelos de agenda e tarefas, e também possa criar o seu modelo
